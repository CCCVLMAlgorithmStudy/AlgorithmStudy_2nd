* 풀이

영어로 끝말잇기를 하는 단어들의 배열이 있을때 몇번째 반복에서 몇번째 사람이 틀렸는지 [번호, 차례 ] 배열로 반환한다.

1. 단어 배열에서 하나씩 꺼내 이전 끝말잇기가 진행되는 지를 확인
2. 이전에 나온 단어인지 확인
3. 둘다 통과면 지금 단어를 이전단어배열에 추가
4. 둘중 하나라도 실패하면 현재 사람과 반복횟수를 반환
